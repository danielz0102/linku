# Linku

Linku is a real-time messaging web application that allows users to connect, chat, and share images with friends.

## Core Features

### User Authentication

- [ ] Users can register a new account
- [ ] Users can log in to their account
- [ ] Users can log out

### User Profiles

- [ ] Users can customize their profile
- [ ] Users can view their own profile
- [ ] Users can view other users' profiles

### Messaging

- [ ] Users can send text messages to another user
- [ ] Users can receive messages from other users
- [ ] Users can view their conversation history with another user
- [ ] Users can see a list of their conversations

### Image Sharing

- [ ] Users can send images in chat conversations
- [ ] Users can view images sent by other users

### Friends List & Online Status

- [ ] Users can add other users to their friends list
- [ ] Users can see which friends are currently online

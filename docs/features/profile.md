# User Profile

User profile includes:

- Profile picture
- Full name
- Username (starts with @, e.g., @john_doe)
- Email address
- Personal bio

All that information can be edited by the user.

## View other users' profiles

When viewing other users' profiles, the following information is displayed:

- Profile picture
- Full name
- Username
- Personal bio
- Status (online/offline)
- Whether is a friend or not

There's a button to add or remove the user from friends list.

# Messaging

## List of chats

Each chat shows:

- Profile picture
- Username
- Last message snippet
- Timestamp of last message
- Unread message count badge (if any)

## Chat

The chat interface includes:

- Header with chat partner's profile picture, username, and online status
- Scrollable message area displaying messages in chronological order
- Input area for sending new messages *and* images

Each message displays:

- Profile picture (for received messages)
- Message content
- Timestamp

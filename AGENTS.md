# Linku

## Introduction

Linku is a messaging web app project for [The Odin Project Curriculum](https://www.theodinproject.com).

The minimum core functionality that this project should have is:

- Authorization
- Sending messages to another user
- Customizing a user profile

## Project Structure

This is a monorepo with the following packages:

- `web`: The frontend web app
- `api`: The backend API
